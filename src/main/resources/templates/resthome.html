REST Home Page

Kaikki Kyselyt

Method: GET Endpoint: /kyselyt

JSON Example:

[
	{
		kysely_id: 1,
		nimi: "Kulttuurivinkkaus-kysely Syksy 2020",
		kuvaus: "Tällä kyselyllä selvitetään tuttujen suosittelemia tärppejä, joilla voi laajentaa tajuntaansa."
	},
	{
		kysely_id: 2,
		nimi: "Koirakysely",
		kuvaus: "Tällä kyselyllä kysellään koirista"
	}
]
