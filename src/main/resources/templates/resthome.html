*** REST Home Page ***

* GET-metodit *

Hae kaikki kyselyt (/kyselyt):

JSON Example: /kyselyt

[
	{
		"kysely_id": 1,
		"nimi": "Kurssipalautekysely",
		"kuvaus": "Tällä kyselyllä selvitetään opiskelijoiden kokemuksia kurssista X."
	}
]

Hae kysely kysely_id:llä (/kyselyt/{kysely_id}):

JSON Example: /kyselyt/1

{
	"kysely_id": 1,
	"nimi": "Kurssipalautekysely",
	"kuvaus": "Tällä kyselyllä selvitetään opiskelijoiden kokemuksia kurssista X."
}

* POST-metodit *

Tallenna vastaus tietylle kysymykselle (/kyselyt/{kyselyid}/kysymykset/{kysymys_id}/vastaus):

JSON Example: /kyselyt/1/kysymykset/2/vastaus

Tallenna tietyn kyselyn vastaukset (/kyselyt/{kyselyid}/vastaukset2):

JSON Example: /kyselyt/1/vastaukset2

	Whitelabel Error Page
